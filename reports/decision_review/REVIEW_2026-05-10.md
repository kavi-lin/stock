## 2. 觀察到的 Patterns

### Pattern A: 「觀望/CANCEL → 錯過熱門 industry 上漲」 [N=59, 信心: 高]
- **觀察**：deep-dive miss=59 中，**樣本 rationale 全部為「觀望/CANCEL → 實際 +X%, 錯過上漲」格式**（10/10 抽樣命中）。重複 ticker：AMD 4/4 全 miss、TSLA 3/3 全 miss、AAPL 3/3 全 miss、VRT 3/3 全 miss、MU 5/7 miss。極端值：AMD 2026-04-15 觀望→+76.35%；MU 2026-04-21 觀望→+66.19%；TSLA 2026-04-07 觀望→+18.79%。
- **與 industry_top_30pct 結合**：top=True miss_rate 0.634 (45/71) vs top=False 0.382 (13/34)。**差距 25.1 pp**，N 充足。
- **root cause 假設 1**：Phase 4 投票機制在熱門 industry 中過度保守。Trader lane 多筆 rationale 顯示「entry 條件達成 / TP 觸發」但 committee 仍 CANCEL → committee 與 trader lane 結論脫鉤。
- **root cause 假設 2**：估值 / Burry / Red Team lane 對 Mature lifecycle 熱門名擔心過高，壓過 Technical / Sentiment 多頭訊號。需要進一步抽 agent_breakdown 才能確認（目前 90/112 未寫入 agent_count，見 Pattern D）。
- **驗證資料**：需要 deep-dive raw `agent_breakdown` 對應每筆 CANCEL 的 lane confidence + dissent 紀錄。

### Pattern B: 「Communication Equipment / Auto-Mfg / Consumer Electronics 100% miss」 [N=11 合計, 信心: 中]
- **觀察**：CSCO/HPE/LITE 5/5 miss、TSLA 3/3 miss、AAPL 3/3 miss，三個 industry 加總 11 筆全部 miss，全屬 top_30% 熱門桶。
- **root cause 假設 1**：mega-cap 加 Mature lifecycle 在當前 cycle phase（4 月中下旬）被普遍判 CANCEL，但實際是 broadening rally 受惠者 — 模型對「成熟大盤股能參與廣度行情」缺反映。
- **root cause 假設 2**：可能是 Burry lane（價值 / 反轉導向）對這三個 industry 估值過高判定主導投票。需 agent_breakdown 確認。
- **驗證資料**：需 N≥10 / industry × agent_breakdown 才能精準歸因。

### Pattern C: 「Sector-scan RISK_ON + defensive skew → 91% hit」 [N=11, 信心: 中]
- **觀察**：sector-scan tuning_hooks `(regime=RISK_ON, rating_skew=defensive)` 11 筆中 hit=10、miss=1。
- **root cause 假設 1**：sector-scan 在多頭環境下「保守 skew」反而是抓對 — 它沒被 regime 帶偏，挑出真正強的 sector。或反向解讀：rating_skew=defensive 計算口徑寬鬆，在多頭裡命中率被誇大（樣本選擇偏差）。
- **驗證資料**：拆解 sector-scan 的 hit 判準（n_hot/n_warm/n_cold 是否真的對映實際 sector return），目前只有「6/11 對 ≈ 隨機」這種 rationale，無法分辨真 hit。

### Pattern D: 「Deep-dive 80% (90/112) 缺 agent_count / decisive_agent」 [N=90, 信心: 高 (資料缺口本身)]
- **觀察**：`tuning_hooks.agent_count == 0` 占 90/112，集中在 2026-04-17 之後。早期（4/7~4/16）有 agent_count=3-4 的 22 筆。

- **root cause 假設 2**：4/17 之後 protocol run 沒持續寫 agent_breakdown 進 session export — 上游 protocol 自身 regression。
- **影響**：所有「decisive_agent x verdict」的 ratio 都建立在 22 筆 sample 上（Fund/News/Tech 各 4-10），不足以下結論。Pattern A/B 的歸因必須等這個資料缺口補上。

### Pattern E: 「news-digest 59% n/a (缺 macro_delta)」 [N=13, 信心: 中 — 資料完整性]
